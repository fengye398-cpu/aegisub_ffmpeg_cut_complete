🎥 第一部分：FFmpeg 环境准备

什么是 FFmpeg？

FFmpeg 是一个开源的音视频处理工具，本软件依赖它来完成：

  • ✅ 视频片段切割
  • ✅ 音频提取
  • ✅ 格式转换
  • ✅ 视频压缩和编码
  • ✅ 字幕嵌入

为什么要配置环境变量？

配置后，本软件才能在任意位置调用 FFmpeg 进行视频处理。

⚠️ 不配置环境变量，软件将无法正常工作！

📥 下载 FFmpeg

步骤 1：访问官网

打开浏览器，访问：https://ffmpeg.org/download.html

步骤 2：选择 Windows 版本

  • 点击 Windows 图标下的链接
  • 推荐选择 gyan.dev 或 BtbN 提供的版本
  • 下载 ffmpeg-release-full.7z

步骤 3：解压文件

将下载的文件解压到合适位置，例如：

C:\Program Files\ffmpeg

⚠️ 路径不要包含中文或特殊字符

⚙️ 配置环境变量（图形界面方法）

步骤 1：打开系统属性

  • 右键点击 此电脑（或"我的电脑"）
  • 选择 属性
  • 点击左侧 高级系统设置
  • 在"系统属性"窗口中，点击 环境变量 按钮

步骤 2：编辑 Path 变量

  • 在"系统变量"区域找到 Path 变量
  • 选中后点击 编辑
  • 点击 新建
  • 输入 FFmpeg 的 bin 目录路径：
C:\Program Files\ffmpeg\bin

点击 确定 保存所有窗口

✅ 验证 FFmpeg 配置

步骤 1：打开命令提示符

  • 按 Win + R 打开运行窗口
  • 输入 cmd 并回车

步骤 2：检查版本

在命令行中输入：

ffmpeg -version

✅ 成功的输出示例

ffmpeg version 6.0-full_build Copyright (c) 2000-2023 the FFmpeg developers ...

✓ 看到类似输出说明配置成功！

❌ 失败的输出示例

'ffmpeg' 不是内部或外部命令...

✗ 出现此错误说明配置失败，请重新检查路径

📦 第二部分：脚本安装

⚠️ 重要：脚本必须放在正确的位置

本软件是 Aegisub 的自动化脚本（.lua 文件），必须放在 Aegisub 的特定目录下才能被识别。

步骤 1：找到 Aegisub 安装目录

  • 右键点击 Aegisub 桌面快捷方式
  • 选择 打开文件位置
  • 这就是 Aegisub 的安装根目录

步骤 2：定位脚本目录

在 Aegisub 安装目录下，逐级打开文件夹：

Aegisub 根目录 └── automation └── autoload ← 脚本需要放在这里

完整路径示例：

C:\Program Files\Aegisub\automation\autoload

或

D:\Program Files\Aegisub\automation\autoload

📦 脚本安装步骤（续）

步骤 3：复制脚本文件

  • 将 ffmpeg_cut_complete.lua 脚本文件复制到 autoload 文件夹
  • 同时也将complete_exporter.exe 复制到 autoload 文件夹
  • 保持文件名不变

步骤 4：重启 Aegisub

  • 关闭 Aegisub（如果正在运行）
  • 重新启动 Aegisub
  • 点击菜单：自动化
  • 检查是否出现 导出选中字幕行对应的音频视频

✅ 安装成功标志

在 Aegisub 的自动化菜单中看到脚本名称，说明安装成功！

❌ 如果菜单中没有显示

  • 检查文件是否放在正确的 autoload 目录
  • 确认文件扩展名是 .lua
  • 检查是否有错误提示（查看 Aegisub 日志）
  • 确保所有依赖文件都已复制

🎯 软件功能介绍

核心功能

本软件是一个 Aegisub 自动化脚本，可以根据字幕时间轴自动切割视频：

  • 📌 按字幕切割：根据字幕文件自动切割视频片段
  • ⚡ 快速导出：支持快速编码模式和重新编码模式
  • 🎬 自动合并：可选择将片段自动合并成完整视频
  • 📝 字幕生成：为每个片段和合并视频生成对应字幕
  • 🎨 参数自定义：支持调整编码质量、分辨率等

适用场景

  • 🎓 制作语言学习材料（按句切割）
  • 📺 提取视频精彩片段
  • 🎵 制作配音素材
  • ✂️ 批量处理视频内容

  • 启动 Aegisub

  • 加载或创建字幕文件
  • 如果有现成字幕和视频，则应先加载字幕再加载视频！！！
  • 如果有现成字幕和视频，则应先加载字幕再加载视频！！！
  • 如果有现成字幕和视频，则应先加载字幕再加载视频！！！

步骤 2：选择要导出的字幕行

  • 在字幕列表中，按住 Ctrl 或 Shift 选择多行
  • 或使用 Ctrl + A 全选

  • 点击菜单：自动化 → 字幕切割导出工具
  • 或使用快捷键（如果已配置）

⚙️ 导出设置详解

步骤 4：配置导出参数

在弹出的对话框中设置：

  • 输出目录：选择片段保存位置
  • 文件命名：

  • 导出模式：快速编码模式 或 重新编码模式

  • 检查设置无误后，点击 开始导出

🎬 连续切割模式详解

什么是连续切割模式？

连续切割模式会从第一条选中字幕的开始时间到最后一条选中字幕的结束时间，切割出一个完整的连续片段，包含中间所有内容（即使没有字幕）。

与片段模式的区别

【片段模式示例】 选中字幕：第1条(2s-5s)、第3条(10s-15s)、第5条(20s-25s) 输出文件：01.mp4(3秒)、02.mp4(5秒)、03.mp4(5秒) 字幕内容：只有3条选中的字幕 【连续模式示例】 选中字幕：第1条(2s-5s)、第5条(20s-25s) ← 只需选首尾 输出文件：01.mp4(23秒，从2s到25s的完整内容) 字幕内容：自动包含时间范围内的所有字幕（5条）

核心特点

  • ✅ 自动收集字幕：自动提取时间范围内的所有字幕
  • ✅ 完整内容：保留中间所有视频内容（包括空白部分）
  • ✅ 相对时间轴：字幕从00:00:00开始，与视频完美对齐
  • ✅ 简化操作：只需选中首尾字幕即可

📝 连续切割模式使用步骤

步骤 1：选择首尾字幕

  • 在字幕列表中选中第一条需要的字幕
  • 按住 Ctrl，点击最后一条需要的字幕
  • 也可以选中中间的字幕，但只有首尾时间起作用

步骤 2：运行连续切割模式

  • 点击菜单：自动化 → 导出选中字幕行对应的音频视频 → 3. 连续切割模式
  • 软件会自动收集时间范围内的所有字幕
  • 弹出对话框显示：收集到 X 条字幕

步骤 3：设置编码参数

  • CRF值：0-51，推荐18-28（默认24）
  • 编码预设：veryfast/fast/medium/slow（默认veryfast）
  • 点击确定开始处理

步骤 4：查看输出文件

输出目录/ ├── 01.mp4 （连续视频片段） ├── 01.mp3 （连续音频片段） ├── 01.srt （相对时间字幕，从00:00:00开始） ├── 视频名_video.mp4 （与01.mp4相同） ├── 视频名_audio.mp3 （与01.mp3相同） └── 视频名_video.srt （与01.srt相同）

🎯 连续切割模式适用场景

📺 完整片段提取

需要提取某个时间段的完整视频内容，包括对话和间隙

🎓 学习材料制作

制作包含完整语境的学习视频，保留对话之间的停顿

🎬 剧情片段

提取连续剧情片段，保持故事完整性

🎤 对话场景

保留对话的自然节奏和停顿

使用建议

  • 💡 完整场景：适合提取完整场景，不适合跳跃式片段
  • 💡 时间范围选择：只需选中首尾字幕，软件自动处理中间内容
  • 💡 字幕精准度：字幕时间轴自动转换为相对时间，与视频完美对齐
  • 💡 质量控制：使用自定义CRF和preset参数控制输出质量

注意事项

  • ⚠️ 连续模式会保留从首字幕到尾字幕的所有视频内容，文件体积可能较大
  • ⚠️ 字幕必须完全在时间范围内（开始时间 >= 首字幕开始 且 结束时间 <= 尾字幕结束）
  • ⚠️ 即使只选中首尾2条字幕，也会自动收集中间所有符合条件的字幕
  • ⚠️ 如果时间范围内没有任何字幕（极少情况），会提示错误

🎓 跟读练习模式详解

什么是跟读练习模式？

跟读练习模式（Shadowing Mode）专为语言学习设计，将每个字幕片段重复播放N次，并在片段之间插入停顿时间，方便学习者跟读模仿。

核心功能

  • 🔁 片段重复：每个句子可重复播放1-10次
  • ⏸️ 智能停顿：自动插入停顿时间供跟读
  • 📝 字幕控制：可选择停顿时是否显示字幕
  • 🎵 双格式支持：支持视频模式和纯音频模式

工作流程示例

原始句子：「こんにちは」(1秒) 跟读模式输出（重复3次，重复间停顿，停顿2倍）： [播放] こんにちは (1秒) [停顿] 显示字幕 (2秒) ← 给学习者跟读时间 [播放] こんにちは (1秒) [停顿] 显示字幕 (2秒) ← 给学习者跟读时间 [播放] こんにちは (1秒) [停顿] 显示字幕 (2秒) ← 给学习者跟读时间 [播放] 下一个句子...

⚙️ 跟读练习模式参数

基础参数

参数 说明 默认值 推荐值
播放次数 每个片段重复播放的次数 3次 2-3次
重复间停顿 是否在重复之间插入停顿 启用 启用

停顿设置

参数 说明 默认值
停顿模式 倍数模式 或 固定秒数模式 倍数模式
停顿倍数 句子时长的倍数（倍数模式） 1.5倍
停顿秒数 固定停顿时长（固定模式） 2.5秒
停顿时显示字幕 停顿期间是否显示字幕 启用

📝 跟读练习模式使用步骤

步骤 1：选择字幕

  • 在 Aegisub 中选中要练习的字幕行
  • 可以选择单行或多行

步骤 2：启动跟读模式

  • 点击菜单：自动化 → 导出选中字幕行对应的音频视频 → 4. 跟读练习

步骤 3：配置参数

  • 播放次数：建议2-3次
  • 停顿倍数：初学者用2倍，熟练者用1.5倍
  • 停顿时显示字幕：初学者启用，进阶者禁用
  • CRF质量：视频模式可设置（默认24）

步骤 4：查看输出

输出目录/ ├── 视频名_shadowing_video.mp4 （视频模式） └── shadowing_audio.mp3 （音频模式） └── shadowing_subtitles.srt （字幕文件）

🎯 跟读练习模式适用场景

🗣️ 口语练习

反复听同一句话，模仿发音和语调

👂 听力训练

重复播放加深理解，提高听力水平

📚 语言学习

制作个性化学习材料，随时随地练习

🎭 配音练习

模仿角色台词，练习配音技巧

使用建议

  • 💡 初学者：播放3次 + 停顿2倍 + 显示字幕
  • 💡 进阶者：播放2次 + 停顿1.5倍 + 不显示字幕
  • 💡 纯音频模式：适合通勤路上练习，节省空间
  • 💡 视频模式：可以观察口型和表情，学习更全面

停顿倍数选择

  • 🔹 1.5倍：快速练习，适合熟悉的内容
  • 🔹 2倍：标准练习，适合大多数场景
  • 🔹 2.5倍+：慢速练习，适合初学者或复杂句子

🎛️ 重新编码参数说明

CRF 值（质量控制）

  • 18-20：极高质量，文件较大
  • 21-23：高质量，适合存档
  • 24-26：良好质量，适中体积（推荐）
  • 27-30：中等质量，文件较小

数值越小，质量越高，体积越大

Preset（编码速度）

  • ultrafast/veryfast：极快，质量稍低
  • faster/fast：较快，质量良好
  • medium：平衡（默认）
  • slow/veryslow：很慢，质量最优

速度越慢，压缩效率越高

❓ 常见问题解答（FFmpeg）

问题 1：提示"未找到 FFmpeg"

  • ✓ 检查 FFmpeg 是否正确安装
  • ✓ 检查环境变量路径是否正确（必须是 bin 目录）
  • ✓ 重启 Aegisub 软件
  • ✓ 在命令行测试 ffmpeg -version

问题 2：配置后仍不可用

  • ✓ 关闭所有命令行窗口后重试
  • ✓ 重启电脑使配置生效
  • ✓ 检查是否配置在"系统变量"而不是"用户变量"

❓ 常见问题解答（软件使用）

问题 1：导出失败或卡住

  • ✓ 检查视频文件路径是否包含中文或特殊字符
  • ✓ 确保有足够的磁盘空间
  • ✓ 检查输出目录是否有写入权限
  • ✓ 尝试使用管理员权限运行 Aegisub

问题 2：音画不同步

  • ✓ 使用重新编码模式而不是快速编码模式
  • ✓ 确保原视频本身音画同步
  • ✓ 检查字幕时间轴是否准确

问题 3：处理速度很慢

  • ✓ 切换到快速编码模式
  • ✓ 重新编码模式选择 faster 或 veryfast preset
  • ✓ 关闭其他占用资源的程序

📁 输出文件结构说明

输出目录结构示例

视频名称_20250103_120000/ （根目录，带时间戳） ├── 视频名称_video.mp4 （合并后的完整视频） ├── 视频名称_audio.mp3 （合并后的完整音频） ├── 视频名称_video.srt （视频版字幕） ├── 视频名称_audio.srt （音频版字幕） └── 视频名称/ （片段目录） ├── 1.mp4 （视频片段1） ├── 1.mp3 （音频片段1） ├── 1.srt （字幕片段1） ├── 2.mp4 ├── 2.mp3 ├── 2.srt └── ...

文件说明

  • 视频片段：根据每行字幕切割的独立视频
  • 音频片段：对应的音频文件
  • 字幕片段：单个片段的字幕文件
  • 合并文件：所有片段按顺序合并的完整文件

💡 使用建议与最佳实践

提高效率的建议

  • 📌 字幕整理：导出前检查字幕时间轴是否准确
  • 📌 命名规范：使用有意义的文件名，便于后续管理
  • 📌 批量处理：一次性选择所有需要的行，避免重复操作
  • 📌 备份原文件：导出前备份原始视频和字幕

质量与速度的权衡

  • 🎯 快速预览：快速编码模式 + 默认参数
  • 🎯 日常使用：重新编码模式 + CRF 24 + medium preset
  • 🎯 高质量存档：重新编码模式 + CRF 20 + slow preset
  • 🎯 网络分享：重新编码模式 + CRF 26-28 + faster preset
