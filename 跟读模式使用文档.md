# 跟读练习模式使用文档

## 功能简介

跟读练习模式（Shadowing Mode）是一个专为语言学习设计的视频/音频导出功能。它将视频或音频按字幕片段切割，每个片段自动重复播放多次，并在片段之间插入停顿时间，方便学习者进行跟读练习。

## 适用场景

- 外语口语练习（跟读训练）
- 听力训练（重复听同一句话）
- 发音模仿练习
- 语言学习材料制作

## 核心功能

### 1. 片段重复播放
- 每个字幕片段可重复播放1-10次
- 适合反复听同一句话，加深记忆

### 2. 智能停顿插入
- 在片段之间自动插入停顿时间
- 给学习者留出跟读/思考的时间
- 支持两种停顿模式：
  - **倍数模式**：停顿时长 = 句子时长 × 倍数（如1.5倍、2倍）
  - **固定模式**：每次停顿固定秒数（如2.5秒）

### 3. 字幕显示控制
- 可选择停顿期间是否显示字幕
- 启用：停顿时显示字幕，方便对照学习
- 禁用：停顿时不显示字幕，增加练习难度

### 4. 双格式支持
- **视频模式**：输出带字幕的视频文件（.mp4）
- **音频模式**：输出音频文件（.mp3）+ 独立字幕文件（.srt）

## 参数说明

### 基础参数

| 参数 | 说明 | 默认值 | 范围 |
|------|------|--------|------|
| **播放次数** | 每个片段重复播放的次数 | 3次 | 1-10次 |
| **重复间停顿** | 是否在重复播放之间插入停顿 | 启用 | 启用/禁用 |

### 停顿设置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **停顿模式** | 倍数模式或固定秒数模式 | 倍数模式 |
| **停顿倍数** | 句子时长的倍数（倍数模式） | 1.5倍 |
| **停顿秒数** | 固定停顿时长（固定模式） | 2.5秒 |
| **停顿时显示字幕** | 停顿期间是否显示字幕 | 启用 |

### 视频质量参数（仅视频模式）

| 参数 | 说明 | 默认值 | 推荐值 |
|------|------|--------|--------|
| **CRF质量** | 视频编码质量（越小越好） | 24 | 18-28 |
| **编码预设** | 编码速度 | veryfast | veryfast/fast/medium |

### 调试参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **手动关闭CMD窗口** | 导出完成后保持CMD窗口打开 | 禁用 |

## 使用步骤

### 1. 准备工作
- 在Aegisub中打开视频/音频文件
- 加载对应的字幕文件（.ass/.srt）
- 确保字幕时间轴已对齐

### 2. 选择片段
- 在字幕编辑区选中要导出的字幕行
- 可以选择单行或多行
- 支持不连续选择

### 3. 启动导出
- 点击菜单：`自动化` → `导出选中片段`
- 选择导出模式：`跟读练习`

### 4. 设置参数
根据学习需求调整参数：

**初学者推荐设置**：
- 播放次数：3次
- 停顿倍数：2倍
- 停顿时显示字幕：启用

**进阶练习设置**：
- 播放次数：2次
- 停顿倍数：1.5倍
- 停顿时显示字幕：禁用

### 5. 开始导出
- 点击确定开始处理
- 等待导出完成
- 输出文件保存在原视频同目录下

## 输出文件说明

### 视频模式输出
```
[原文件名]_YYYYMMDD_HHMMSS/
├── [原文件名]_shadowing_video.mp4    # 跟读练习视频
└── shadowing_debug.txt                # 调试信息（可选）
```

### 音频模式输出
```
[原文件名]_YYYYMMDD_HHMMSS/
├── shadowing_audio.mp3                # 跟读练习音频
├── shadowing_subtitles.srt            # 字幕文件
└── shadowing_debug.txt                # 调试信息（可选）
```

## 常见问题

### Q: 为什么实际时长和理论时长有差异？
A: FFmpeg切割视频时需要对齐关键帧，会产生微小误差（通常<100ms）。这是正常现象，不影响使用。

### Q: 纯音频模式和视频模式有什么区别？
A:
- 纯音频模式：只加载音频文件，输出音频+字幕
- 视频模式：加载视频文件，输出带字幕的视频

### Q: 如何选择停顿倍数？
A:
- 1.5倍：快速练习，适合熟悉的内容
- 2倍：标准练习，适合大多数场景
- 2.5倍+：慢速练习，适合初学者或复杂句子

### Q: 停顿时是否应该显示字幕？
A:
- 启用：适合初学者，可以对照学习
- 禁用：适合进阶练习，增加难度

### Q: 导出的视频文件很大怎么办？
A: 调整CRF质量参数：
- CRF 18：高质量，文件大
- CRF 24：标准质量（推荐）
- CRF 28：较低质量，文件小

## 技术说明

- 视频编码：H.264
- 音频编码：AAC（视频）/ MP3（纯音频）
- 字幕格式：SRT（内嵌或独立）
- 时间精度：毫秒级
- 支持的输入格式：MP4, MKV, AVI, MP3, WAV等
