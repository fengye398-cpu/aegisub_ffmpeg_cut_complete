# aegisub_ffmpeg_cut_complete
1.这是一个根据字幕切割音/视频的软件;This is software that splits audio and video based on subtitles.

2.要把ffmpeg三件套加入到系统变量

3.尽量用我提供的ffmpeg版本，切割速度会更快，不保证用其它版本影响切割速度

4.教程视频:https://www.bilibili.com/video/BV1LrjB62EZT/

5.参考项目:https://github.com/tumuyan/video-shuffler-for-aegisub
